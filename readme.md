<h2>🛠️ Installation Steps:</h2>

<p>1. Clone</p>

```
girt clone git@github.com:chmenegatti/hashing.git
```

<p>2. Enter folder</p>

```
cd hashing
```

<p>3. Install Dependencies</p>

```
cargo build
```

<p>4. Run</p>

```
cargo run
```
